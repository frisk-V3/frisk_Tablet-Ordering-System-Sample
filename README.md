このプロジェクトは、タブレットを使用したシンプルなオーダーシステムのサンプルです。ユーザーは、メニューを選択して注文をカートに追加し、最終的に注文を送信することができます。レスポンシブデザインを採用しており、タブレットやモバイルデバイスでの利用に最適化されています。

特徴

タブレット専用UI: タブレットに最適化されたインターフェース。

リアルタイム注文管理: メニューをタップするだけで注文がカートに追加されます。

簡単な注文送信: 注文の内容を確認した後、簡単に注文を送信できます。

レスポンシブ対応: 小さなデバイスでも利用可能。

使用技術

HTML5: レスポンシブデザインに対応したHTML

CSS3: スタイリングとアニメーション

JavaScript (Vanilla JS): オーダー機能のロジック（メニュー選択、カート表示、合計計算）

インストールとセットアップ

リポジトリをクローンする

git clone https://github.com/frisk-V3/frisk_Tablet-Ordering-System-Sample.git

使用方法

メニューから商品を選択します。

画面右側の注文パネルに追加され、合計金額が計算されます。

注文が完了したら、「注文を送信」ボタンをクリックして注文を送信します。

開発者向け
ファイル構成

index.html: メインのHTMLファイル

style.css: 全体のスタイルシート

script.js: JavaScriptによる注文管理のロジック

カスタマイズ方法

メニューの更新: メニュー項目を追加/変更したい場合、HTML内の.menu-itemクラスを更新してください。

価格やカロリーの変更: 各メニュー項目のdata-priceやdata-cal属性を変更して、価格やカロリーを変更できます。
